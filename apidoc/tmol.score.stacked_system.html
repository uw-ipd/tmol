Bases: tmol.score.factory.Factory

Score graph component describing stacked system’s “depth” and “size”.

A score graph is defined over a set of independent system layers. The number layers defines the stacked “depth”, and the maximum number of atoms per layer defines the system “size”. Each layer is defined over the same maximum number of atoms, but systems may have varying number of null atoms.

Atom indices are defined by a layer index, atom index (l, n) tuple.

stack_depth¶

The system stack depth, l.

system_size¶

The maximum number of atoms per layer, n.
