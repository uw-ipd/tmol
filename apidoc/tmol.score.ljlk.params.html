Container for global/type/pair parameters, indexed by atom type name.

Param resolver stores pair parameters for a collection of atom types, using a pandas Index to map from string atom type to a resolver-specific integer type index.

Convert array of atom type names to parameter indices.

pandas.Index.get_indexer only operates on 1-d input arrays. Coerces higher-dimensional arrays, as may be produced via broadcasting, into lower-dimensional views to resolver parameter indices.
